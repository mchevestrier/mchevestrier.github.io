# Logic Symbols

|                      | Conjunction  | Disjunction | Negation   |
| -------------------- | ------------ | ----------- | ---------- |
| Keyword              | AND          | OR          | NOT        |
| C (logical)          | &&           | \|\|        | !          |
| C (bitwise)          | &            | \|          | ~          |
| Prolog               | ,            | ;           | +          |
| Mathematics          | ∧            | ∨           | ¬          |
| Boolean algebra      | \*           | +           | ‾A or A′   |
| Set Theory           | Intersection | Union       | Complement |
| Set Theory (symbols) | ∩            | ∪           | ∁          |
